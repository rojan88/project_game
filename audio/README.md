# Audio Placeholders (Milestone 4)

Drop basic `.wav` files in this folder to enable the M4 audio pass:

- `ui_click.wav` - button presses, menu actions
- `hit_enemy.wav` - melee hit connect
- `dodge.wav` - player dodge
- `ability.wav` - companion/special and stage-clear accent

The game safely falls back to silence if these files are missing.
